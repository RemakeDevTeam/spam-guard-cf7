海外からの大量迷惑問い合わせを排除するプラグインです。お問い合わせ内容に日本語が含まれていない場合は送信されません。

== Description ==

Contact Form 7のフォームに日本語入力チェック機能を追加するプラグインです。
指定したフィールドに日本語が含まれていない場合、フォームの送信をブロックします。

主な機能：
* お問い合わせ本文の日本語チェック
* 差出人名の日本語チェック（オプション）
* 管理画面からのフィールドID設定
* Contact Form 7との完全な互換性

== Installation ==

1. 管理画面の「プラグイン」→「新規追加」から「アップロード」を選択
2. このプラグインのZIPファイルを選択してアップロード
3. プラグインを有効化
4. Contact Form 7の管理メニューから「お問合せ制限」を選択して設定

== Frequently Asked Questions ==

= Contact Form 7は必要ですか？ =

はい、このプラグインの使用にはContact Form 7が必要です。

= どの言語をチェックしますか？ =

ひらがな、カタカナ、漢字のいずれかが含まれているかをチェックします。
